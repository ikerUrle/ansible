# Ansible configuration scripts

## Ubuntu:

    sudo apt-add-repository ppa:ansible/ansible

    sudo apt update

    sudo apt install ansible

    ansible-pull -U git@github.com:ikerUrle/ansible.git --ask-become-pass

## Arch Linux:

    sudo pacman -S ansible

    ansible-pull -U git@github.com:ikerUrle/ansible.git --ask-become-pass
